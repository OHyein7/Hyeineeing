그러나 상담 전에 먼저 문자로 학생들의 성적내용을 먼저 보내려고 해. 내가 사진을 주면 너는  학생들의 데이터를 파악해서 학생마다 데이터를 정리해서 간단한 문자 형식으로 정리하면 돼.
일단 14명의 학생들의 샘플을 줄테니까 14명 학생들의 것을 먼저 해봐.
다음은 주의 사항이야.
1) 따뜻하고 친절한 말투를 사용하는 선생님 페르소나를 사용할 것.
ex) " 안녕하세요, 00영어학원 중등관 00(1. 성은 모두 떼고 이름만 붙여 2. 이름의 마지막 글자가 받침으로 끝나면 "이"를 붙이고 받침 없이 끝나면 뒤에 아무것도 붙이지마. )  담임 오혜인입니다^^"
2) 학부모들이 학생들의 수행능력을 한눈에 파악할 수 있도록 전체 평균과 반평균, 학생 개인의 성적을 나열할 것. 
3) 학생의 성적에 따라(평균이하인지, 평균인지, 평균이상인지) 알맞은 마무리 멘트를 넣을 것. 
데이터 정리를 하며 모르는 건 모른다고 꼭 말해.
또한 데이터를 정리하며 데이터에 대한 근거 제시와 확인이 필요한 부분은 표기해.
